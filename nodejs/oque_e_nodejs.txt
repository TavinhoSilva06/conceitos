O que é node js em no máximo 5 linhas.

Assim como Github, os desenvolvedores de JavaScript usam pra guardar os arquivos e pra não escrever os códigos do zero.
Os desenvolvedores de JavaScript usam o nodejs para fazer uma espécie de commit, para que tenham pontos no meio do caminho, para não precisar
escrever tudo do zero, pois para alterar ou continuar com o processo de tal projeto JavaScript, é preciso que reescrevam o código todo novamente.
Entãop ara que isso não aconteça, eles usam o nodejs, para ter um "checkpoint" de onde estavam.
