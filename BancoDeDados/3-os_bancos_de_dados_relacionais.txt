Quais os bancos de dados relacionais existem?

MySQL, PostgreSQL, MariaDB, Microsoft SQL Server e Oracle.

MySQL:
MySQL é um Banco de Dados relacional (RDBMS – Relational Database Management Systems) com um
modelo de cliente servidor.

PostgreSQL:
PostgreSQL é um gerenciador de banco de dados relacionados que otimiza muito o trabalho
de quem precisa administrar informações nesses níveis. 

MariaDB:
MariaDB foi desenvolvida pelos mesmos desenvolvedores de MySQL no ano de 2009, ela veio como proposta substituir o MySQL, então fazer
uma versão melhorada do próprio. MariaDB tem diversas vantagens quando é utilizada, ela é uma ferramenta de código aberto, pode 
desenvolver criptografia em rede, aplicativo e servidor, funciona em qualquer base de dados na nuvem e tem um custo de hospedagem 
baixo para os padrões.

SQL server: 
O SQL Server Integration Services é uma plataforma para criar soluções de integração de dados de alto desempenho,
incluindo pacotes que fornecem processamento de ETL (extração, transformação e carregamento) para Data Warehousing.

Oracle:
A Oracle Corporation é uma empresa multinacional de tecnologia e informatica norte-americana, especializada no 
desenvolvimento e comercialização de hardware e softwares e de banco de dados.
